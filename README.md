# cccrrraaaiiiggg.com

Personal website of Craig Johnson — designer and maker of things.

## Structure

- **`index.html`** — Homepage with rotating image gallery
- **`about.html`**, **`publications.html`**, **`links.html`** — Static pages
- **`projects/`** — Individual project pages (music, film, photography, installation, illustration, and more)
- **`sites/`** — Standalone interactive sites and games hosted under the same domain
- **`css/style.css`** — Global styles
- **`js/sidebar.js`** — Sidebar web component and project index
- **`js/lightbox.js`** — Image lightbox
